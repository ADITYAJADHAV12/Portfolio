# Aditya Portfolio Website

A modern, responsive personal portfolio website built to showcase skills, projects, education, and contact information in a clean and professional way. This project is designed as a static front-end portfolio using HTML, CSS, and JavaScript, which is a common structure for portfolio templates hosted on GitHub or deployed as static sites [web:32][web:33].

## Resume Setup

If the resume button is not showing the file, make sure:
- Your resume file is added inside the correct folder
- The file name matches exactly with the link in your HTML
- The path is correct, for example:
  ```html
  <a href="./assets/resume/Aditya_Resume.pdf" target="_blank">View Resume</a>
  ```

File paths and exact naming matter for static portfolio websites because the browser only opens files that exist at the specified location [web:32].
